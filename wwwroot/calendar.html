📅 Calendar Help

📅 Calendar Views

  • Month View: See the full month with all events
  • Week View: Focus on a single week's schedule
  • Day View: Detailed view of a single day with time slots

➕ Adding Events

  • Click "Add Event" button or click on any day
  • Fill in event title, date, and optional time
  • Choose event category (Interview, Deadline, etc.)
  • Add description for more details

🔄 Recurring Events

  • Check "Make this a recurring event" for regular events
  • Choose frequency: Daily, Weekly, or Monthly
  • Set an end date to limit how long events repeat
  • Perfect for weekly deadlines or regular meetings

📊 Event Categories

  • Interview: Job interviews and assessments
  • Deadline: Application deadlines and important dates
  • Follow-up: Reminders to follow up on applications
  • Networking: Career fairs, meetups, and networking events
  • Meeting: General meetings and appointments
  • Other: Any other calendar events

💾 Export & Navigation

  • Use arrow buttons to navigate between months/weeks/days
  • Click "Today" to jump to the current date
  • Export your calendar events as CSV for backup
  • Click on events to edit or delete them
💡 Pro Tip: Use recurring events for weekly application deadlines and set up interview reminders to stay organized during your job search!
